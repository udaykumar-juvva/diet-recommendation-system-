#  Diet Recommendation System 🥗
A high-performance web application built with Streamlit and Python to provide personalized diet protocols.

## 🚀 Features
- **Bio-metric Analysis:** Real-time BMI calculation with interactive Plotly gauges.
- **Goal-Oriented:** Supports Weight Loss, Weight Gain, and Muscle Gain protocols.
- **Glassmorphism UI:** Modern "Aurora Liquid" design with responsive mobile support.
- **PDF Generation:** Downloadable PDF reports using FPDF.

## 🛠️ Tech Stack
- **Frontend:** Streamlit, Custom CSS
- **Visualization:** Plotly
- **Data Handling:** Python
- **Deployment:** Streamlit Community Cloud
